## Run locally
These commands are examples; adapt to your framework (Next.js, CRA, etc.).

1. Clone the repo
```bash
git clone https://github.com/Yogesh7564/portfolio.git
cd portfolio
```

2. Install dependencies
```bash
npm install
# or
yarn
```

3. Run the development server
```bash
npm run dev
# or
yarn dev
```

Open http://localhost:3000 in your browser (or the port configured by your framework).

## Deployment
Deploy to Vercel, Netlify, or GitHub Pages. Example for Next.js on Vercel:
- Connect the repository to Vercel
- Set build command: `npm run build`
- Set output directory (if needed): `.next` (Vercel auto-detects Next.js)

For GitHub Pages (static builds):
- Build with `npm run build`
- Publish the `out` or `build` directory depending on your setup
